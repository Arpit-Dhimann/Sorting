# Java Sorting Algorithms

Welcome to my Java sorting repository! This project is dedicated to mastering fundamental sorting algorithms, analyzing their logic, and implementing them efficiently.

## Current Progress: Sorting Algorithms

Inside this repository, you will find clean, modular implementations of core sorting techniques:

*   **Bubble Sort** (`Bubble_Sort.java`): A comparison-based sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.
*   **Selection Sort** (`Selection_Sort.java`): An algorithm that sorts an array by repeatedly finding the minimum element from the unsorted part and putting it at the beginning.
*   **Insertion Sort** (`Insertion_Sort.java`): A simple sorting algorithm that builds the final sorted array one item at a time by taking elements from the unsorted list and inserting them into their correct position.
